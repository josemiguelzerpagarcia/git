Examen Git y Github

Alumn@: Alumno JM ZG

  1. ¿Que es un sistema de control de versiones?

    es un software sincroniza las versiones y garantiza que los cambios no entren en conflicto con los cambios de otros usuarios

  2. ¿Cual es la diferencia entre git y github?

    git es la aplicacion en si mientras que github es la zona con todos los repositorios y demas

  3. ¿Cómo creo una rama en git?

    para crear una rama debemos de poner el comando git branch

  4. ¿Cómo clono un repositorio?

    debes de poner en consola: git clone [link del repositorio]

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    debes de poner en consola: git pull

  6. ¿Cómo puedo generar un tag?

    nos vamos a la configuracion de nuestro perfil y en el apartado mas le damos a crear tag

  7. Cómo hago una mezcla de ramas?

    en consola, cambiamos a la rama que queramos mezclar y ponemos git merge-rama feature

  8. ¿Cómo puedo moverme a un commit especifico?

    usando el comando git checkout

  9. ¿Cómo puedo borrar una rama?

    usando el comando git branch -d [nombre de la rama]

  10. ¿Cómo puedo borrar un tag?

    usando el comando git tag -d [nombre del tag]

  11. ¿Cómo puedo ver los cambios que estan por enviarse al repositorio?

    debes de poner en consola: git status

  12. ¿Para que sirve el comando git remote -v?

    se usa para mostrar las URLs de los repositorios remotos asociados con tu repositorio local

  13. ¿Cómo le digo a git cual es mi email y correo?

    debes de poner en consola: git config --global -user.mail "[correo]"

  14. ¿Cuales son los pasos para conectar git con github?

    debes de generar una clave publica y una privada, luego nos copiamos la publica y la pegamos en el apartado de claves en nuestro github

  15. ¿Cómo puedo crear un alias en git?

    usando el comando git config --global alias.nombre-del-alias '[nombre del comando]'
